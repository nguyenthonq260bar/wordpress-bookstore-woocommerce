# The Mountain Book

The Mountain Book is an e-commerce bookstore developed using WordPress and WooCommerce as the final project for the E-Commerce course.

## Technologies

- WordPress
- WooCommerce
- PHP
- MySQL
- HTML, CSS, JavaScript
- Custom Theme: Thefirstdayinyourlife
- Custom Plugin: Company WooCommerce

## Project Structure

```
.
├── wordpress/          # WordPress source code
├── database.sql        # Database backup
└── README.md
```

## Installation

1. Install a local web server (XAMPP, Laragon, MAMP, etc.).
2. Copy the `wordpress` folder into the web server directory.
3. Create a MySQL database.
4. Import `database.sql`.
5. Update the database connection in `wp-config.php` if necessary.
6. Start Apache and MySQL.
7. Open the website in your browser.

## Default Features

- Product management
- Category and tag management
- Shopping cart
- Checkout
- Order management
- Shipping methods
- Payment methods (COD & VietQR)
- Wishlist
- AJAX product search
- Dark / Light Mode
- AI Book Consultant
- Live Chat
